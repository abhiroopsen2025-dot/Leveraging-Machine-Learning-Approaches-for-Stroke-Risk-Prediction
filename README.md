# Leveraging Machine Learning Approaches for Stroke Risk Prediction

## Overview

This project explores the use of machine learning techniques to analyze and predict the risk of stroke based on patient health and demographic data. The primary objective is to identify patterns, understand feature relationships, and build predictive models that can assist in early stroke risk assessment.

## Project Structure

* **01_Stroke_Risk_Prediction_EDA.ipynb** – Exploratory Data Analysis (EDA) of the stroke dataset, including data cleaning, visualization, and insights.
* *(Future notebooks)*

  * **02_Data_Preprocessing.ipynb**
  * **03_Feature_Engineering.ipynb**
  * **04_Model_Training.ipynb**
  * **05_Model_Evaluation.ipynb**

## Dataset

The dataset contains patient information such as:

* Age
* Gender
* Hypertension
* Heart Disease
* BMI
* Average Glucose Level
* Smoking Status
* Work Type
* Residence Type
* Marital Status
* Stroke (Target Variable)

## Objectives

* Perform exploratory data analysis (EDA).
* Identify missing values and outliers.
* Analyze feature distributions and correlations.
* Visualize relationships between patient attributes and stroke occurrence.
* Prepare the data for machine learning models.
* Develop and evaluate predictive models for stroke risk.
